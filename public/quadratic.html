一元二次方程解法详解

系统学习 · 五种解法 · 轻松掌握

ax² + bx + c = 0 (a≠0)

📖 一、什么是一元二次方程

🎯 方程的定义

方程：含有未知数的等式叫做方程。

一元二次方程：只含有一个未知数，未知数的最高次数是2的整式方程。

📐 标准形式

ax² + bx + c = 0 (a≠0)

🎨 各项可视化理解

二次项 ax² 决定抛物线开口方向和宽度
一次项 bx 影响抛物线位置平移
常数项 c 决定抛物线与y轴交点
名称 表达式 示例：2x² - 5x + 3 = 0
二次项 ax² 2x²（系数 a=2）
一次项 bx -5x（系数 b=-5）
常数项 c 3（c=3）
⚠️ 注意：a≠0 是关键条件！如果 a=0，方程就变成了一元一次方程。

🔍 生活中的例子

场景：用 20 米长的篱笆围成一个矩形花坛，要使面积为 24 平方米，边长是多少？

分析：设一边长为 x 米，则另一边为 (20-2x)/2 = 10-x 米

列方程：x(10-x) = 24 → x² - 10x + 24 = 0

✏️ 判断练习

下列哪些是一元二次方程？（点击查看答案）

① x² - 3x + 2 = 0 ✓ 是（标准形式）
② 2x + 5 = 0 ✗ 不是（一次方程）
③ x² + y² = 4 ✗ 不是（二元方程）
④ 0x² + 3x - 1 = 0 ✗ 不是（a=0）
⑤ (x-1)² = 4 ✓ 是（展开后是标准形式）

📌 解法一：直接开平方法

▼

适用条件：形如 x² = k 或 (x+m)² = k 的方程

📝 解题步骤

  1. 确认方程形式符合 x² = k
  2. 开平方得 x = ±√k
  3. 写出两个解

📌 例题

题目：解方程 x² = 16

点击查看解答 ▼

解：

x² = 16

x = ±√16

x = ±4

∴ x₁ = 4，x₂ = -4

⚠️ 易错提醒：开平方时不要漏掉负根！x² = 16 的解是 x = ±4，不是 x = 4。

📌 解法二：配方法

▼

适用条件：任意一元二次方程（特别适用于求最值、证明不等式）

📝 解题步骤（口诀："移、化、配、开"）

移

将常数项移到等号右边

x² + bx = -c

化

二次项系数化为 1

x² + (b/a)x = -c/a

配

两边同加 (b/2)²

x² + (b/a)x + (b/2a)² = -c/a + (b/2a)²

(x + b/2a)² = ...

开

开平方求解

x + b/2a = ±√(...)

  1. 移：将常数项移到等号右边
  2. 化：将二次项系数化为 1（两边同除以 a）
  3. 配：两边同加 (b/2)²，配成完全平方式
  4. 开：开平方求解

📌 例题一

题目：解方程 x² - 6x + 5 = 0

点击查看解答 ▼

解：

第一步（移）：x² - 6x = -5

第二步（化）：二次项系数已经是 1，跳过

第三步（配）：x² - 6x + 9 = -5 + 9

∵ (b/2)² = (-6/2)² = 9

(x - 3)² = 4

第四步（开）：x - 3 = ±2

x - 3 = 2 或 x - 3 = -2

∴ x₁ = 5，x₂ = 1

📌 例题二

题目：解方程 2x² + 8x - 10 = 0

点击查看解答 ▼

解：

第一步（移）：2x² + 8x = 10

第二步（化）：x² + 4x = 5（两边同除以 2）

第三步（配）：x² + 4x + 4 = 5 + 4

∵ (b/2)² = (4/2)² = 4

(x + 2)² = 9

第四步（开）：x + 2 = ±3

x + 2 = 3 或 x + 2 = -3

∴ x₁ = 1，x₂ = -5

⚠️ 易错提醒：配方时，(b/2)² 要同时加到等号两边！

📌 解法三：公式法

▼

适用条件：任意一元二次方程（万能方法）

📐 求根公式

x = -b ± √(b²-4ac)2a

其中 Δ = b² - 4ac 称为判别式

📊 判别式的意义

两个不相等的实数根

抛物线与x轴有两个交点

两个相等的实数根

抛物线与x轴有一个交点（顶点）

无实数根

抛物线与x轴无交点

判别式 Δ 根的情况
Δ > 0 两个不相等的实数根
Δ = 0 两个相等的实数根
Δ < 0 无实数根

📝 解题步骤

  1. 将方程化为标准形式，确定 a、b、c 的值
  2. 计算判别式 Δ = b² - 4ac
  3. 代入求根公式求解

📌 例题一

题目：解方程 2x² + 3x - 2 = 0

点击查看解答 ▼

解：

第一步：确定系数 a = 2，b = 3，c = -2

第二步：计算判别式

Δ = b² - 4ac = 3² - 4×2×(-2) = 9 + 16 = 25 > 0

方程有两个不相等的实数根

第三步：代入公式

x = -3 ± √252×2 = -3 ± 54

x₁ = (-3+5)/4 = 2/4 = 1/2

x₂ = (-3-5)/4 = -8/4 = -2

∴ x₁ = 1/2，x₂ = -2

📌 例题二

题目：解方程 x² - 4x + 4 = 0

点击查看解答 ▼

解：

第一步：确定系数 a = 1，b = -4，c = 4

第二步：计算判别式

Δ = b² - 4ac = (-4)² - 4×1×4 = 16 - 16 = 0

方程有两个相等的实数根

第三步：代入公式

x = -(-4) ± √02×1 = 42 = 2

∴ x₁ = x₂ = 2

⚠️ 易错提醒：代入公式时注意 b 的符号！b = -4 时，-b = -(-4) = 4

📌 解法四：因式分解法

▼

适用条件：方程左边易于分解因式的情况

📝 解题步骤

  1. 移项使右边为 0
  2. 左边分解因式
  3. 令各因式为 0 求解

📌 例题一（提公因式法）

题目：解方程 x² - 5x = 0

点击查看解答 ▼

解：

提取公因式 x：

x(x - 5) = 0

令各因式为 0：

x = 0 或 x - 5 = 0

∴ x₁ = 0，x₂ = 5

📌 例题二（分组分解法）

题目：解方程 x² - 5x + 6 = 0

点击查看解答 ▼

解：

分解因式：x² - 5x + 6 = (x - 2)(x - 3)

(x - 2)(x - 3) = 0

x - 2 = 0 或 x - 3 = 0

∴ x₁ = 2，x₂ = 3

⚠️ 易错提醒：分解因式后，不要忘记令每个因式都等于 0！

📌 解法五：十字交叉法

▼

适用条件：分解形如 ax² + bx + c 的二次三项式

📝 方法原理

将 ax² + bx + c 分解为 (mx + p)(nx + q)

需要满足：

  • m × n = a（二次项系数）
  • p × q = c（常数项）
  • mq + np = b（一次项系数）

📐 十字交叉图示

ax² + bx + c = (mx + p)(nx + q)

💡 关键：找到合适的 m、n、p、q，使交叉相乘后相加等于一次项系数 b

📌 例题一

题目：分解 x² + 5x + 6

点击查看解答 ▼

解：

分析：a=1, b=5, c=6

1 ↘ 2
1 ↗ 3
1×3 + 1×2 = 5 ✓

x² + 5x + 6 = (x + 2)(x + 3)

📌 例题二

题目：分解 2x² - 7x + 3

点击查看解答 ▼

解：

分析：a=2, b=-7, c=3

尝试分解：

2 ↘ -1
1 ↗ -3
2×(-3) + 1×(-1) = -7 ✓

2x² - 7x + 3 = (2x - 1)(x - 3)

📌 例题三（完整求解）

题目：解方程 3x² - 10x + 3 = 0

点击查看解答 ▼

解：

用十字交叉法分解 3x² - 10x + 3：

3 ↘ -1
1 ↗ -3
3×(-3) + 1×(-1) = -10 ✓

3x² - 10x + 3 = (3x - 1)(x - 3)

(3x - 1)(x - 3) = 0

3x - 1 = 0 或 x - 3 = 0

∴ x₁ = 1/3，x₂ = 3

💡 技巧：当 a、c 为正数，b 为正数时，分解后都是正；当 a、c 为正数，b 为负数时，分解后都是负。
⚠️ 易错提醒：交叉相乘方向不要搞反！验证时确保 mq + np = b。

🌍 四、实际应用

📐 应用一：面积问题

问题：用 24 米长的篱笆围成一个矩形花坛，要使面积为 36 平方米，花坛的边长应是多少？

点击查看解答 ▼

解：

设矩形的一边长为 x 米，则另一边长为 (24-2x)/2 = (12-x) 米

根据题意列方程：x(12-x) = 36

化简：-x² + 12x = 36

x² - 12x + 36 = 0

(x-6)² = 0

x = 6

另一边：12 - 6 = 6（米）

答：花坛的边长为 6 米，是一个正方形花坛。

🏀 应用二：抛物问题

问题：一个物体从高处自由下落，下落的高度 h（米）与时间 t（秒）的关系为 h = 5t²。若物体下落了 45 米，求下落时间。

点击查看解答 ▼

解：

根据题意列方程：5t² = 45

t² = 9

t = ±3

∵ 时间不能为负数，∴ t = -3 舍去

答：物体下落 45 米需要 3 秒。

📈 应用三：增长率问题

问题：某厂一月份产值 100 万元，第一季度总产值 364 万元，求二、三月份平均每月的增长率。

点击查看解答 ▼

解：

设平均增长率为 x

一月份：100 万元

二月份：100(1+x) 万元

三月份：100(1+x)² 万元

列方程：100 + 100(1+x) + 100(1+x)² = 364

化简：(1+x)² + (1+x) + 1 = 3.64

设 y = 1+x，则 y² + y + 1 = 3.64

y² + y - 2.64 = 0

解得 y = 1.2 或 y = -2.2

y = 1+x = 1.2，∴ x = 0.2 = 20%

（y = -2.2 < 0，不合题意舍去）

答：平均每月增长率为 20%。

💰 应用四：利润问题

问题：某商场销售一种商品，每件进价 40 元，售价 60 元，每周可卖出 300 件。经调查发现，每降价 1 元，每周可多卖出 20 件。要使每周利润达到 6125 元，应降价多少元？

点击查看解答 ▼

解：

设降价 x 元

降价后每件利润：(60-x) - 40 = (20-x) 元

降价后销量：300 + 20x 件

列方程：(20-x)(300+20x) = 6125

化简：6000 + 400x - 300x - 20x² = 6125

20x² - 100x + 125 = 0

4x² - 20x + 25 = 0

(2x-5)² = 0

x = 2.5

答：应降价 2.5 元。

📋 五、考点与易错

🎯 典型考点

考点一：基础概念

考点 常见题型 解题关键
判断方程类型 给出多个方程，判断哪些是一元二次方程 抓住三个条件：一个未知数、最高次数2、整式方程、a≠0
确定系数 求 ax²+bx+c=0 中 a、b、c 的值 先化为标准形式，注意符号
方程的根 已知根，求方程或系数 根代入方程成立

考点二：解法选择

考点 常见题型 解题关键
选择最优解法 给方程，选择最简便方法 观察方程特征
配方法应用 求最小/最大值、证明不等式 配成 (x+m)²+n 形式
公式法推导 推导求根公式 掌握配方法过程
判别式应用 判断根的个数、求参数范围 Δ>0 两不等根，Δ=0 两相等根，Δ<0 无实根

考点三：韦达定理

考点 常见题型 解题关键
基本应用 求 x₁+x₂、x₁·x₂ 直接代入公式：x₁+x₂=-b/a，x₁x₂=c/a
变形应用 求 x₁²+x₂²、1/x₁+1/x₂ 等 用 x₁+x₂、x₁·x₂ 表示
构造方程 已知两根，构造方程 x²-(x₁+x₂)x+x₁x₂=0
根与系数关系 已知一根，求另一根和参数 联立韦达定理求解

韦达定理：设 x₁、x₂ 是方程 ax²+bx+c=0 的两根，则：

x₁ + x₂ = -b/a     x₁ · x₂ = c/a

💡 几何意义：两根之和等于对称轴位置的两倍

💡 记忆技巧：和用 b，积用 c，都除以 a

考点四：应用题

类型 常见情境 设未知数技巧
面积问题 围栏、地砖、裁剪 设边长为 x
增长率问题 两年增长相同比例 设增长率为 x
利润问题 降价促销、定价策略 设降价/涨价 x 元
传播问题 病毒传播、消息转发 设每次传播 x 人
相遇问题 两人相向而行 设速度/时间为 x

⚠️ 易错点详解

一、概念理解易错

易错点 错误示例 正确理解
二次项系数 认为 x²-2x+1=0 的 a=0 a=1，常数项为 0 时方程仍是二次方程
标准形式 x²=2x+3 直接写 a=1,b=2,c=3 先移项：x²-2x-3=0，a=1,b=-2,c=-3
方程的解 x²=4 的解为 x=2 x=±2，漏掉负根是常见错误
判别式条件 使用韦达定理不考虑 Δ≥0 必须先判断方程有实根

二、计算过程易错

易错点 错误示例 正确做法
移项变号 x²-5=3x → x²+3x-5=0 x²-3x-5=0，移项要变号
配方法漏加 x²+4x+1=0 → x²+4x+4=-1 两边同加 4：x²+4x+4=3
公式代入符号 2x²-3x+1=0 中 b=3 b=-3，负号是系数的一部分
开平方漏根 (x-1)²=9 → x-1=3 x-1=±3，得 x=4 或 x=-2
十字交叉方向 x²+5x+6 中，交叉写反 验证：交叉相乘后相加应等于一次项系数

三、韦达定理易错

易错点 错误示例 正确做法
符号混淆 x²+3x-4=0 中 x₁+x₂=3 x₁+x₂=-b/a=-3，注意负号
使用前提 不验根就使用韦达定理 必须先判断 Δ≥0
构造方程 根为 2、3，方程为 x²+5x+6=0 x²-5x+6=0，两根之和为 -b/a
齐次变形 求 x₁²+x₂² 直接等于 (x₁+x₂)² x₁²+x₂²=(x₁+x₂)²-2x₁x₂

四、应用题易错

易错点 错误示例 正确做法
单位不统一 面积用平方米，边长用厘米 统一单位后再列方程
忽略实际意义 答案 x=-5（边长为负） 检验并舍去不合实际的解
增长率公式错误 增长率为 x，两年后写成 (1+x)² 倍 正确，但注意原始量乘这个倍数
解答不完整 只写 x=5，没有写"边长为5米" 完整作答：答：边长为5米
设问理解错误 求增长率，设了增长人数 设增长率为 x（百分数）

五、书写规范易错

易错点 错误写法 正确写法
解的表示 x=2,-3 x₁=2，x₂=-3 或 x=2 或 x=-3
公式书写 分数线过短，运算顺序不清 用括号明确：(-b±√Δ)/(2a)
解方程格式 缺少"解"字，直接开始 开头写"解："，步骤清晰
检验过程 应用题不写检验 写"经检验，x=-5 不合题意，舍去"

🚀 六、知识拓展

📜 数学史话

一元二次方程的历史渊源...

古巴比伦时期（约公元前2000年）

古巴比伦人已经能够解决一些特殊的一元二次方程问题，他们在泥板上记录了许多涉及面积和边长的问题，这些问题可以转化为一元二次方程。

中国《九章算术》（约公元前1世纪）

中国古代数学名著《九章算术》中已有"开方"的方法，可以求出正根。例如"今有积五万五千二百二十五步，问为方几何"，实际上就是解 x² = 55225。

阿拉伯数学家花拉子米（约公元820年）

阿拉伯数学家花拉子米在其著作《代数学》中，系统地讨论了一元二次方程的解法，给出了求根公式的几何证明。

韦达与韦达定理（16世纪）

法国数学家韦达发现了方程的根与系数之间的关系，即著名的"韦达定理"。这一发现极大地简化了许多代数问题的求解。

📊 知识延伸

与一元二次方程相关的数学知识...

二次函数与一元二次方程

  • Δ > 0：抛物线与 x 轴有两个交点
  • Δ = 0：抛物线与 x 轴有一个交点（相切）
  • Δ < 0：抛物线与 x 轴无交点

高次方程简介

一元三次方程和一元四次方程也有求根公式，但非常复杂。挪威数学家阿贝尔在1824年证明了五次及以上的方程不存在一般的求根公式。

图象法求解

对于无法用公式求解的方程，可以用图象法求近似解：画出函数图像，找与 x 轴交点。

💡 进阶技巧

提升解题能力的技巧...

整体思想

把某个代数式看作整体，简化问题。

例：解方程 (x+1)² - 4(x+1) + 3 = 0

设 y = x+1，则 y² - 4y + 3 = 0

(y-1)(y-3) = 0

y = 1 或 y = 3

x+1 = 1 或 x+1 = 3

x = 0 或 x = 2

换元法

对于特殊形式的方程，用换元法简化。

例：解方程 x⁴ - 5x² + 4 = 0

设 y = x²，则 y² - 5y + 4 = 0

(y-1)(y-4) = 0

y = 1 或 y = 4

x² = 1 或 x² = 4

x = ±1 或 x = ±2

参数问题处理

含参数的方程，需要讨论参数的取值范围。

例：关于 x 的方程 x² - 2mx + m² - 1 = 0 有两个实根，求 m 的范围。

Δ = 4m² - 4(m²-1) = 4 ≥ 0

所以对任意实数 m，方程都有两个实根。
