# 初中二年级英语语法学习网页设计文档

## 项目概述

一个面向初中二年级学生的英语语法学习网页，采用多页面架构，结合知识讲解、混合练习题型和游戏化元素，使用本地存储管理学习进度。

## 技术选型

- **前端**：原生 HTML + CSS + JavaScript（无需框架依赖）
- **存储**：localStorage 本地存储
- **语音**：Web Speech API（浏览器原生支持）
- **风格**：活泼卡通风格，明亮色彩

## 页面结构


## 页面设计详情

### 首页

**布局结构：**

- **顶部仪表盘**
  - 用户等级和积分（进度条展示）
  - 今日学习时长/题目数
  - 连续学习天数（带火焰图标）

- **中间语法模块卡片**
  - 7个语法模块卡片网格（3列布局）
  - 每张卡片：卡通图标 + 语法名称 + 进度条 + 星星评分
  - 卡片颜色随进度变化（灰色→浅色→明亮）

- **底部导航栏**
  - 首页、排行榜、设置、帮助

**交互：**
- 点击卡片进入对应语法讲解页
- 长按卡片可快速开始练习

### 语法讲解页面

**布局结构：**

- **顶部**
  - 面包屑导航：首页 > 语法模块名称

- **左侧内容区（70%宽度）**
  - 语法概念卡通插图（如时间轴表示时态）
  - 语法规则讲解（简洁语言+表格）
  - 典型例句（中英对照，可点击朗读）
  - 常见错误提醒（警示图标标注）
  - 与其他语法的对比

- **右侧快捷栏（30%宽度）**
  - 快速开始练习按钮
  - 本节要点速记卡片
  - 相关语法推荐链接

- **底部**
  - 上一节/下一节导航
  - 返回首页按钮

**交互：**
- 例句点击播放语音（Web Speech API）
- 要点卡片点击展开详细解释

### 练习页面

**布局结构：**

- **顶部进度条**
  - 当前题目进度（如 3/10）
  - 正确率实时统计（绿色对勾/红色叉子图标）

- **题目区**
  - 选择题：题目 + 四个选项卡片（点击选中，即时反馈）
  - 填空题：题目带空格 + 输入框 + 提交按钮
  - 判断题：题目 + 正确/错误两个大按钮
  - 连词成句：打乱的单词卡片，拖拽排序

- **答案反馈**
  - 正确：绿色动画 + 欢呼音效 + 积分+10
  - 错误：红色提示 + 正确答案解析 + 鼓励语

- **底部操作**
  - 上一题/下一题按钮
  - 查看解析按钮
  - 返回按钮

**游戏元素：**
- 连续答对3题触发"连击"奖励（双倍积分）
- 每完成一个模块获得成就徽章

## 数据存储方案

### localStorage 结构

```javascript
// 用户进度数据
localStorage.setItem('userProgress', JSON.stringify({
  level: 5,
  totalPoints: 1250,
  streakDays: 7,
  lastStudyDate: "2026-03-28"
}));

// 各语法模块进度
localStorage.setItem('grammarProgress', JSON.stringify({
  'present-perfect': {
    viewed: true,
    completed: 8,
    correct: 6,
    stars: 2
  },
  // ... 其他模块
}));

// 成就徽章列表
localStorage.setItem('achievements', JSON.stringify([
  'first_study', 'streak_3', 'perfect_10'
]));

// 用户设置
localStorage.setItem('settings', JSON.stringify({
  soundEnabled: true,
  theme: 'cartoon'
}));
```

### 数据同步

- 每次操作自动保存到 localStorage
- 页面加载时读取并恢复状态

## 语法模块内容

1. **现在完成时 (Present Perfect Tense)**
   - 构成：have/has + 过去分词
   - 用法：表示过去发生但对现在有影响的动作

2. **过去进行时 (Past Continuous Tense)**
   - 构成：was/were + doing
   - 用法：表示过去某时刻正在进行的动作

3. **情态动词 (Modal Verbs)**
   - can, could, may, might, must, should, will, would
   - 各情态动词的用法和区别

4. **被动语态 (Passive Voice)**
   - 构成：be + 过去分词
   - 各种时态的被动形式

5. **形容词/副词比较级和最高级 (Comparison)**
   - 构成规则：-er/-est, more/most
   - 不规则变化

6. **不定式和动名词 (Infinitive & Gerund)**
   - to do vs doing 的区别
   - 常见搭配

7. **连词和状语从句 (Conjunctions & Adverbial Clauses)**
   - 时间、条件、原因、结果等从句
   - 常用连词
